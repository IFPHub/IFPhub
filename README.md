Descripción General

IFP Platform es una aplicación web diseñada para estudiantes de IFP, donde pueden interactuar con la comunidad académica publicando proyectos, creando o inscribiéndose en eventos, solicitando citas con secretaría, accediendo a reuniones grabadas y más. El objetivo es centralizar recursos y dinamizar la vida académica.

Tecnologías Principales

Frontend: React + Next.js

Backend / Base de datos: Supabase

Despliegue: Vercel

Funcionalidades

Publicación de proyectos por estudiantes.

Creación y registro en eventos.

Solicitud de citas con secretaría.

Visualización de reuniones grabadas.

Acceso autenticado mediante Supabase Auth.
